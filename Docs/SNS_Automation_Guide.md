# 🚀 [B2B] SNS 마케팅 자동화 서비스 시작 가이드

안녕하세요! **ChoiGPT 마케팅 자동화 솔루션**을 선택해 주셔서 감사합니다.
본 가이드는 대표님의 소중한 비즈니스를 24시간 잠들지 않는 AI 홍보 로봇이 관리할 수 있도록 설정하는 과정을 담고 있습니다.

누구나 5분이면 따라 할 수 있도록 상세히 설명해 드릴 테니, 천천히 함께 진행해 주세요!

---

## 1. 🛠️ 준비 단계 (사전 체크리스트)

자동화 시스템이 대표님의 인스타그램에 게시물을 올리기 위해 다음 두 가지가 필요합니다.

1.  **인스타그램 비즈니스/크리에이터 계정**: 개인 계정이라면 `설정 -> 계정 유형 및 도구 -> 프로페셔널 계정으로 전환`을 눌러주세요. (무료입니다!)
2.  **페이스북 페이지 연결**: 인스타그램 설정에서 `센터 -> 연결된 환경 -> 페이스북 페이지 연결`을 완료해 주세요.

---

## 2. 🔐 계정 정보 입력 및 보안 안내

전용 관리자 페이지에서 **인스타그램 아이디와 비밀번호**를 입력해 주셔야 합니다.

> [!IMPORTANT]
> **왜 아이디/비밀번호가 필요한가요?**
> 공식 API 연동 과정이 복잡하여, 초기 세팅 시 AI 에이전트가 직접 보안 토큰을 발급받기 위해 필요합니다. 
> 입력하신 정보는 **AES-256 군사급 암호화 기술**로 보호되며, 세팅 완료 후에는 저희도 직접 열람할 수 없는 형태로 안전하게 관리됩니다.

**입력 방법:**
1.  전달드린 [마케팅 관리자 페이지] 링크에 접속합니다.
2.  `신규 고객 등록` 섹션으로 이동합니다.
3.  상호명과 업체 설명을 상세히 적어주세요. (AI가 이 내용을 바탕으로 분석 리포트를 만듭니다!)
4.  인스타그램 계정 정보를 정확히 입력하고 `등록` 버튼을 누릅니다.

---

## 3. 📊 상권 분석 리포트 확인하기

등록 후 약 5~10분 정도 기다리시면, AI가 대표님의 업체를 위해 다음 내용을 분석합니다.

-   **온라인 인지도**: 현재 네이버나 인스타그램에서 우리 가게가 어떻게 비춰지고 있는지?
-   **주변 경쟁사 현황**: 주변 업체들은 어떤 전략으로 홍보하고 있는지?
-   **마케팅 전략 제언**: 어떤 키워드와 컨셉으로 홍보해야 매출이 오를지?

관리자 페이지의 `분석 리포트 보기` 버튼을 통해 실시간으로 확인하실 수 있습니다.

---

## 4. 📅 자동화 운영 및 관리

세팅이 완료되면 다음과 같이 작동합니다.

-   **업로드 주기**: 기본적으로 **24시간마다 1회** 최적의 시간에 포스팅됩니다. (조정이 필요하시면 말씀해 주세요!)
-   **콘텐츠 내용**: 대표님이 선택하신 마케팅 이론(STP, AIDA 등)에 따라 전문적인 문구와 이미지가 생성됩니다.
-   **모니터링**: 대표님은 평소처럼 장사에 집중하세요. 홍보는 AI가 알아서 수행합니다.

## 💡 자주 묻는 질문 (FAQ)

**Q. 비밀번호를 나중에 바꿔도 되나요?**
A. 네! 시스템이 세팅된 이후에는 비밀번호를 바꾸셔도 서비스가 유지되도록 설계하고 있습니다. 다만, 큰 변경이 있을 경우 다시 업데이트가 필요할 수 있습니다.

**Q. 홍보글이 마음에 안 들면 어떡하죠?**
A. 관리자 페이지에서 '적용 마케팅 방법론'을 변경해 보세요. 분위기가 완전히 다른 콘텐츠가 생성됩니다.
